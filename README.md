# 📊 E-Learners Database — SQL Project

A MySQL project analyzing purchase data from an online learning platform that sells digital courses to learners across different countries.

---

## 📖 Introduction

This project focuses on analyzing purchase data from an online learning platform using *MySQL*. The platform offers various digital courses, and learners from different countries can purchase them.

The project is designed to understand:
- Database creation
- Table relationships
- Constraints (Primary Key, Foreign Key)
- SQL Operations (DDL, DML, Queries)

---

## 🎯 Objectives

- ✅ Create a structured database (e_learnersdb)
- ✅ Establish relationships using foreign keys
- ✅ Understand learner behavior
- ✅ Identify popular courses and categories
- ✅ Apply SQL concepts like GROUP BY, HAVING, JOIN, etc.

---

## 📌 Key SQL Concepts Applied

| Concept | Description |
|--------|-------------|
| PRIMARY KEY | Uniquely identifies each record |
| FOREIGN KEY | Links tables and enforces referential integrity |
| JOIN | Combines data from multiple tables |
| GROUP BY | Groups rows for aggregate analysis |
| HAVING | Filters grouped results |
| COUNT / SUM / AVG | Aggregate functions for insights |
