# SQL-Project
This project focuses on designing and implementing a social media database using MYSQL. This system stores about users, posts, followers, comments and likes. The system stores information about users, posts, followers, comments and likes. 
The objective is to understand:
•	Database creation
•	Table relationships
•	Constraints (Primary Key, Foreign Key)
•	SQL Operations (DDL, DML, Queries)
Objectives: 
•	To create a structured database 
        create database social_media;
•	To establish relationships using foreign keys
•	To perform data insertion and retrieval.
•	To apply SQL concepts like GROUP BY, HAVING, JOIN etc.
SQL Operations Performed:
	DDL COMMANDS:
Create database, Create Table, Alter table, Drop table
	DML COMMANDS:
Insert, Delete, Select
